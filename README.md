# ReelsMakerPro [mod by llimonix]

![Превью приложения](resources/banner.png)

Упрощённая сборка ReelsMaker Pro в один EXE и установщик **без установки Python, Inno Setup и PyInstaller** на вашем компьютере.

---

## Оригинал и версия

- **Оригинальный проект:** [nellimonix/ReelsMakerPro](https://github.com/nellimonix/ReelsMakerPro)
- **Версия приложения в этой модификации:** **1.1.1** (та же, что в оригинале на момент форка)
- **Что изменено в модификации:** один скрипт скачивает встроенный Python, ставит зависимости, собирает EXE и установщик; установщик создаётся без Inno Setup (копия в папку пользователя + ярлыки); обход падения PyInstaller при сборке с torch/whisper.

---

## Как пользоваться упрощённой версией

### Вариант 1: Вы хотите собрать EXE и установщик сами

1. Скачайте или клонируйте этот репозиторий.
2. Запустите в корне проекта **один файл:**  
   **`Собрать_установщик.bat`**
3. Дождитесь окончания (при первом запуске — дольше: качается Python, зависимости, ffmpeg, yt-dlp, идёт сборка).
4. В папке **`dist`** появятся:
   - **ReelsMakerPro-Setup.exe** — отдавайте пользователям (установщик в один клик);
   - **ReelsMakerPro.exe** — портативный запуск без установки.

Ничего заранее ставить не нужно: ни Python, ни Inno Setup, ни PyInstaller. Нужны только Windows и интернет.

**Сборка на GitHub:** вкладка **Actions** → workflow **"Build installer"** → Run workflow → скачать артефакт **ReelsMakerPro-Setup**.

---

### Вариант 2: Вы конечный пользователь (вам дали установщик)

1. Запустите **ReelsMakerPro-Setup.exe**.
2. При необходимости выберите папку установки (или оставьте по умолчанию).
3. Дождитесь окончания установки — появятся ярлыки (Пуск, рабочий стол).
4. Запускайте **ReelsMaker Pro** как обычную программу.

Python, FFmpeg и yt-dlp ставить не нужно — всё уже внутри приложения.

## Что умеет приложение

ReelsMaker Pro — обработка видео, вертикальные форматы (Reels/TikTok), субтитры (в т.ч. через Whisper), загрузка на YouTube с автозаполнением метаданных.

- **Обработка:** фильтры, цвет, скорость, зум, обрезка, оверлеи, субтитры (SRT или автогенерация Whisper).
- **Форматы:** конвертация, в т.ч. 1080×1920 (Reels/TikTok), размытый фон, автообрезка.
- **YouTube:** загрузка, приватность, расписание, ИИ-метаданные.
- **Дополнительно:** пакетная обработка, скачивание с YouTube (yt-dlp), предпросмотр.

Подробный список возможностей — в разделе [Функциональность](#функциональность) ниже.

---

## Функциональность

### Обработка видео
Фильтры (черно-белое, сепия, инверсия, размытие, пикселизация, VHS), цвет (яркость, контраст, насыщенность), трансформации, оверлеи (изображения, GIF), скорость, зум, форматы, аудио, субтитры (SRT или Whisper).

### Reels/TikTok
Вертикальный формат 1080×1920, размытый фон, автообрезка.

### ИИ и YouTube
Автогенерация метаданных, распознавание речи (Whisper), загрузка на YouTube с настройкой приватности и расписания.

### Другое
Пакетная обработка, предпросмотр, загрузка видео с YouTube, удаление метаданных.

## Для разработчиков: запуск из исходников

Если нужен Python и ручная настройка (как в оригинале):

```bash
git clone https://github.com/nellimonix/ReelsMakerPro.git
cd ReelsMakerPro
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

**FFmpeg и yt-dlp:** либо добавьте в PATH (см. [оригинальный README](https://github.com/nellimonix/ReelsMakerPro)), либо положите `ffmpeg.exe`, `ffprobe.exe`, `yt-dlp.exe` в папку `bin/` в корне проекта.

**YouTube API:** [instruction.md](instruction.md).

## Сборка EXE вручную (без упрощённого батника)

Если уже есть Python и зависимости:

```bash
python build_exe.py
```

В `dist/` появится **ReelsMakerPro.exe**. Установщик отдельно:

```bash
python build_setup_exe.py
```

Или полная цепочка с установщиком: `build.bat` (нужен установленный [Inno Setup](https://jrsoftware.org/isinfo.php) для классического установщика) или упрощённый путь — **`Собрать_установщик.bat`** (Inno не нужен).
